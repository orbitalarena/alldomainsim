C++ Engine & MC Server

Build: cd build && cmake .. && ninja mc_engine
Serve: cd visualization/cesium && python3 serve.py 8000
MC Server: node mc_server.js 8001 (enables C++ MC from the Scenario Builder)
Replay: ./bin/mc_engine --replay --scenario <path> --seed 42 --max-time 600 --output replay.json
Batch MC: ./bin/mc_engine --scenario <path> --runs 1000 --seed 42 --output results.json
